# AI Assets README

This folder stores reusable AI governance and agent prompt assets for this repository.

## Files
- `master-context.md`: project context, architecture guardrails, and policy baseline.
- `cleanup-policy.md`: mandatory cleanup checklist before PR.
- `viseo-*.md`: role prompts for active project agents.

## Usage
1. Load `master-context.md` for any significant task.
2. Run the relevant agent prompt from this folder.
3. Apply `cleanup-policy.md` before requesting review.
4. Update matching memory file in `engineering/ai-memory/` with verified facts.
