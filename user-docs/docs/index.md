# Documentation utilisateur de Belotable

Bienvenue dans la documentation utilisateur de Belotable.

Belotable permet d'organiser un concours de belote, de saisir les scores et de visualiser les résultats.

## Ce que vous trouverez ici

- Comment installer l'application
- Comment utiliser l'application pour gérer un concours

## Public cible

Cette documentation s'adresse aux organisateurs de tournois et aux joueurs qui souhaitent utiliser l'application rapidement.
