# dev-scripts

Les scripts présents dans ce dossier sont destinés à être utilisés par les développeurs pour faciliter certaines tâches liées au développement. Ils sont conçus pour être exécutés localement sur la machine du développeur.
Ces scripts ont aussi été créés pour réduire l'utilisation d'agents IA pour des tâches qui peuvent être accomplies de manière déterministe ou répétitive, afin de réduire la consommation de tokens et les coûts associés à l'utilisation de Github Copilot.
